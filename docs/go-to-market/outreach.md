# Outreach Assets

## Short Invite

I am opening the beta for **DevAssemble**, a terminal-first launcher for existing Next.js applications. It scans the current project, provisions the required infrastructure, and deploys without requiring manual provider dashboard work.

I am looking for teams willing to:

- run one guided launch with me
- run one self-serve follow-up launch later
- give direct feedback on clarity, trust, and failure recovery

## Feedback Prompts

- Did the product make the launch path clear before execution began?
- Did the execution view feel trustworthy while the run was in progress?
- If something failed, was the next action obvious?
- Which parts felt polished?
- Which parts still felt risky or unfinished?
