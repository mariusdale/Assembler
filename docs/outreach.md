# Private Beta Outreach Assets

## Short Invite

Hi - I am running a private beta for **DevAssemble**, a terminal-first launcher for existing Next.js SaaS apps. It scans your project, provisions the infra it detects, and deploys without dashboard hopping.

I am looking for indie hackers willing to do:

- one guided launch with me
- one self-serve retry later
- blunt feedback on clarity, trust, and failure recovery

If that sounds useful, I would love to onboard you.

## Walkthrough Script

Target length: 60-90 seconds.

1. Start in an existing Next.js app directory.
2. Run `devassemble setup` to validate provider credentials.
3. Run `devassemble` and show the TUI launch briefing:
   - detected providers
   - readiness state
   - risks and expected outputs
4. Start the launch and narrate the execution console:
   - current phase
   - active task
   - recent run activity
5. End on the launch summary:
   - preview URL
   - repo URL
   - warnings
   - next steps

## Beta Feedback Prompts

- Did you understand what DevAssemble was doing before it started?
- Did you trust the execution view while it was running?
- If something failed, did the product make the next action obvious?
- What felt magical?
- What felt risky or unclear?
